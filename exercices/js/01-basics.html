Exercise 2: Arithmetic Operations

Practice basic arithmetic operations with variables.

// TODO: Create two variables 'a' and 'b' with numbers // TODO: Calculate and store the sum in a variable called 'sum' // TODO: Calculate and store the difference in a variable called 'difference' // TODO: Calculate and store the product in a variable called 'product' // TODO: Calculate and store the quotient in a variable called 'quotient' // TODO: Calculate and store the remainder in a variable called 'remainder' // TODO: Print all results using console.log()

Exercise 4: Type Conversion

Practice converting between different data types.

// TODO: Create a string variable 'numberString' with value "42" // TODO: Convert it to a number and store in 'number' // TODO: Create a number variable 'ageNumber' with value 25 // TODO: Convert it to a string and store in 'ageString' // TODO: Create a string variable 'truthString' with value "true" // TODO: Convert it to a boolean and store in 'truth' // TODO: Print all variables and their types using console.log()
Solution:
let numberString = "42"; let number = Number(numberString); let ageNumber = 25; let ageString = String(ageNumber); let truthString = "true"; let truth = Boolean(truthString); console.log("Original string:", numberString, typeof numberString); console.log("Converted number:", number, typeof number); console.log("Original number:", ageNumber, typeof ageNumber); console.log("Converted string:", ageString, typeof ageString); console.log("Original string:", truthString, typeof truthString); console.log("Converted boolean:", truth, typeof truth);

Exercise 5: Math Operations

Use Math object methods to perform calculations.

// TODO: Create a variable 'decimal' with value 3.7 // TODO: Round it to the nearest integer using Math.round() // TODO: Round it up using Math.ceil() // TODO: Round it down using Math.floor() // TODO: Calculate the square root of 16 using Math.sqrt() // TODO: Calculate 2 to the power of 8 using Math.pow() // TODO: Find the absolute value of -15 using Math.abs() // TODO: Print all results using console.log()
Solution:
let decimal = 3.7; let rounded = Math.round(decimal); let roundedUp = Math.ceil(decimal); let roundedDown = Math.floor(decimal); let squareRoot = Math.sqrt(16); let power = Math.pow(2, 8); let absolute = Math.abs(-15); console.log("Original:", decimal); console.log("Rounded:", rounded); console.log("Rounded up:", roundedUp); console.log("Rounded down:", roundedDown); console.log("Square root of 16:", squareRoot); console.log("2 to the power of 8:", power); console.log("Absolute value of -15:", absolute);
